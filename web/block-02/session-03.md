# Session 3 — Webots Tutorials 1–7

**Duration: 3 hours · Not graded**

---

## Goal

Before the first assignment, you work through the official [Webots User Guide Tutorials 1–7](https://cyberbotics.com/doc/guide/tutorials). These tutorials guide you step by step through modelling a simple robot and writing your first controller.

By the end of Session 3, you will:

- [ ] Know how to open and navigate Webots
- [ ] Have built and run a simple robot model yourself
- [ ] Understand how a controller communicates with a simulated robot

---

## What to Do

Open the [official Webots tutorials](https://cyberbotics.com/doc/guide/tutorials) and work through tutorials 1–7 in order. Each tutorial builds on the previous one — do not skip steps.

!!! tip "Work at your own pace"
    The tutorials are self-paced. If you do not finish all seven during the session, continue at home. Bring open questions to Session 4.

---

## Webots Interface — Quick Reference

| Area | Function |
|---|---|
| 3D view | Main simulation window — watch and interact with the world |
| Scene tree | Hierarchical list of all objects in the world |
| Text editor | Built-in editor for writing and editing controllers |
| Console | Shows print output from your controller |
| Toolbar | Play / pause / reset the simulation |

---

## Troubleshooting

If Webots does not start or behaves unexpectedly, check the [Webots FAQ](https://cyberbotics.com/doc/guide/general-faq) or ask in the next session. Common issues are GPU driver problems and missing installation of the Visual C++ Redistributable (Windows only).
