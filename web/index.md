# Autonomous Logistics Systems and IoT

Welcome to the self-study companion for the Master's course **Autonomous Logistics Systems and IoT** at OTH Amberg-Weiden.

!!! info "About this site"
    This site contains self-study material, assignment descriptions, and reference content for all course blocks. It accompanies the live sessions — not a replacement for attending class.

---

## Course Structure

| Block | Topic | Format |
|---|---|---|
| **Block 1** | Organisation & AGV Foundations | Lecture + student research |
| **Block 2** | Classical AGVs with Webots | Hands-on simulation |
| **Block 3** | Autonomous AGVs with ROS | Demo + guided activities |
| **Block 4** | AGV Simulation | Plant Simulation assignment |
| **Assessment** | Business Case | Semester break submission |

## Assessment

| Component | Weight | Timing |
|---|---|---|
| In-semester work (Blocks 2–4) | 70 % | During semester |
| Business Case | 30 % | Semester break |
