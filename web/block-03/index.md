# Block 3 — Autonomous AGVs with ROS

**Sessions 9–10 · Duration: approx. 2 weeks · 3 hours per session**

This block introduces the technical foundations of autonomous robot navigation and the middleware that ties them together: the Robot Operating System (ROS). You will observe a real AGV — the TurtleBot3 — build a map of its environment entirely on its own, and then navigate it autonomously to a goal you specify.

No programming is required. Your task is to run provided launch files, observe the results, and explain what you see.

---

## Software — ROS Noetic

All activities run on **pre-configured virtual machines** (VMs) provided in the lab. ROS Noetic and all required packages are pre-installed — no setup is needed on your part.

!!! warning "Use the lab VMs"
    Do not attempt to install ROS on your personal computer for these sessions. The pre-configured VM is the only supported environment.
