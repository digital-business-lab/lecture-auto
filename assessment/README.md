# Assessment

The **Business Case** is the primary mandatory assessment of this course. It is the format most aligned with the BWL target group's competencies and career context.

## Folder Structure

| Folder | Contents |
|---|---|
| `business-case/` | Template document and grading rubric for the business case |

## Business Case Components

| Component | Description | Status |
|---|---|---|
| Market research | AGV systems, vendors, market trends | ⏳ Template pending (AP9) |
| Business case calculation | ROI, payback period, TCO, cost-benefit analysis | ⏳ Template pending (AP9) |
| Requirements specification (Lastenheft) | Structured procurement requirements | ⏳ Template pending (AP9) |
| Utility value analysis (Nutzwertanalyse) | Weighted scoring of AGV alternatives | ⏳ Template pending (AP9) |
| Grading rubric | Evaluation criteria and point allocation | ⏳ Pending (AP9) |
