# lecture-auto

Accompanying material for the Master's course **Autonomous Logistics Systems and IoT** at the Digital Business Lab.

The course covers the use of Automated Guided Vehicles (AGVs) in intralogistics, with hands-on work in Webots, ROS/Gazebo, and custom simulation.
